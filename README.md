#WINE EXCHANGE

An e-commerce website which allows users to buy and review wines.

Currently hosted on: https://cs2team15.cs2410-web01pvm.aston.ac.uk

Features:
- INDEX PAGE
- CHATBOT
- FOOTER
- ABOUT US
- SEARCH PAGE/WINE PAGE
- WINE INFO
- FIND A GIFT
- CONTACT US
- SEARCH FUNCTION
- LOGIN PAGE
- SIGN UP PAGE
- FORGOT YOUR PASSWORD
- ACCOUNT PAGE
- BASKET
- CHECKOUT
- ADMIN PAGE
- INVENTORY PAGE
- NEW WINE PAGE
- EDIT WINE

Tech Stack:
- HTML/CSS/JS
- PHP

INDEX PAGE:
- introduces the user to the website and welcomes them.
- has a small banner which acts as an advert. showing recommended wines to the user.
- has a section that reveals a users recently viewed wines.
- includes website reviews. writing them and displaying them to users.
- has a mini section that has our frequently asked questions.

CHATBOT:
- there is a chatbot named edward.
- Edward assists users with wine recommendations and provides additional information when requested.

FOOTER:
- includes a subscription option. allowing the user to subscribe to our newsletter.
- has many different links including a shortcut to browsing all wines and some other pages including.
about us, FAQ, Contact us, shipping & delivery, terms and conditions.

ABOUT US:
- about us is a page designed to educate the user on what we stand for as a business.
- the last section mentions our wine collection and has a button to navigate the user to our wine collection.

SEARCH PAGE/ WINES PAGE:
- this page acts as our wines page.
- it has many different cards for each individual wine.
- a user can immediately add it to the basket if its in stock.
- if its not in stock the user is notified in a user friendly way.
- if they simply click on the card they are navigated to a new page with additional information on their selected wine.
- it also has a filter which allows the user to sort their wines further.

WINE INFO:
- this page shows further detail on each individual wine.
- a user can add or remove from the basket.
- a user can add or remove from the wishlist.
- a user can press share. which copies the link for the user allowing them to then share the wine.
- the page includes multiple images of each wine.
- there is a small section on each.
- there is a section towards the bottom that both displays product reviews and allows the user to leave a review.

FIND A GIFT:
- this page is designed for users who intend to buy a wine on behalf of someone else.
- it includes a small quiz that intends to narrow down the different wines to find a specific wine based on their answers.
- once the quiz is completed the recommended wines are displayed to the user allowing them to take their pick.

CONTACT US:
- this page displays the location of where wine exchange is located (dummy location).
- it has a section that allows a user to send a direct email/message to our admins.
- these messages can involve things such as website complaints, order mistakes etc.

SEARCH FUNCTION:
- this page works the same way as wines page.
- it allows the user to search for a specific wine by different features such as wine name, wine type (a further extension on the filter).

LOGIN PAGE:
- the login page has a simple design intended to allow the user to easily login without any issues.
- we have a login with google feature which allows the user to quickly login without wasting time.
- we have a 2FA feature which acts as additional security to ensure only authorised users can login to their account.
- as an extra security feature. if a user enters the incorrect credentials too many times they are kicked out the website for 30 seconds.

SIGN UP PAGE:
- the signup page is very similar to the login page and has a very simple design to ensure ease of use.
- it also has a google sign up feature.
- one issue with signing up with google is that not all data is added during the process.
to tackle this there is an additional page to ensure we collect all necessary information (e.g. age as you must be 18+ to purchase a wine).

FORGOT PASSWORD:
- if a user no longer remembers their password they can simply request a password reset.
- if that email exists a reset email is sent to them allowing them to change their password.

ACCOUNT PAGE:
- this page is displayed after a user logs in.
- it displays their current details and allows the user to logout, edit or delete their data.
- user also has the option to add a profile picture to their account.
- if the user is also an admin then a button will appear which allows them navigation to the admin page.
- towards the bottom their is a table which shows a customers transaction history.
- on this table a customer can request a refund/return as long as they have a valid reason (return request must be approved or declined by the admin).

BASKET:
- the basket appears empty initially.
- once a product is added it should appear in the basket where the user can change the quantity (based on the given stock).
- there is a remove button that obviously removes a wine from the basket (this is also reflected in the stock).

CHECKOUT PAGE:
- the checkout page is designed to complete the transaction/purchase.
- if a user is not logged in (checking out as a customer) then they are required to enter all of their details.
- if the user is logged in then all thats required is their payment type (apple pay or credit/debit).
- after successfully completing their purchase they are given a message box confirming their order.

ADMIN PAGE:
- if a user is logged in as an admin they can access this page.
- they can filter through all the current users (admin, adminpending and customer).
- they can also search through the customers based on their customer ID.
- an admin can edit or delete a users details.
- they can also directly change a users password.
- the transaction section shows all current transactions and allows the admin to make changes where required (e.g. adding a shipping number).
- the returns section shows all the current return requests. each one can be approved or rejected by the admin. until then its in a state of pending.

INVENTORY PAGE:
- this page shows the stock of all the current wines.
- towards the top it has a brief summary of the crucial info (e.g. total active wines, total bottles in stock, shows the amount of wines below a certain amount and shows any wines that are currently out of stock).
- the page has a full wine listing which allows actions for each wine (editing or deleting a wine).
- there is also a button to allow new wines to be added to our stock listing.

NEW WINE PAGE:
- this page displays each necessary field for adding a wine (e.g.name, region, price, stock, image etc).
- once confirmed the wine will be added to the wine listings.

EDIT WINE PAGE:
- this page works the exact same way as the new wines page. only its purpose is to edit our existing wines.
