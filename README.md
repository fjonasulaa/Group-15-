#WINE EXCHANGE

An e-commerce website which allows users to buy and review wines.

Currently hosted on: https://cs2team15.cs2410-web01pvm.aston.ac.uk/frontend/html/index.php

Features:
- 
- Wine Catalogue
- Basket Feature
- Wishlist
- Product Reviews
- User Registration
- Account Dashboard
- Edit/Delete Acoount
- Admin Roles
- Guest Checkout
- Age Verification
- Stock Management
- Gift Finder Quiz
- Dark Mode

Tech Stack:
- 
- HTML/CSS/JS
- PHP
